# minecraftstackfinder
Just a handy tool to find the number of items in a stack + remainders :)

# How to use
1) Run the program.
2) Input the number of items in the input field.
3) Input whether it is a special item ('y' for yes, and 'n' for no).
3.5) if yes, input the stack size for the special item. For example Ender Pearls are 16.
4) The program will return the number of full stacks and remaining items, this text will be in green.
